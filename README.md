# Retail Sales Performance Analytics Project

## Project Overview

This project analyzes retail sales performance using the Sample Superstore dataset. The objective is to identify trends in sales, profitability, customer segments, product categories, and regional performance to support data-driven business decisions.

## Tools Used

* Microsoft Excel
* SQL
* Power BI
* Tableau
* GitHub

## Business Objectives

* Analyze sales performance across regions.
* Identify top-performing product categories.
* Evaluate profitability by category and sub-category.
* Assess the impact of discounts on profit.
* Generate actionable business recommendations.

## Dataset

Source: Sample Superstore Dataset

Records: 9,994

Features include sales, profit, quantity, discounts, customer segments, regions, categories, and sub-categories.
