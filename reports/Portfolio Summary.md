# Project

Retail Sales Performance Dashboard

# Tools
  * Power BI
  * Power Query
  * DAX
  * Data Modelling
    
# Business Questions Answered
  * Which category drives the most revenue?
  * Which products are unprofitable?
  * Which region generates the most profit?
  * How do discounts impact profitability?
  * Which customer segment creates the most value?

# Key Findings
  * Technology is the top-performing category.
  * Consumer segment generates the highest revenue.
  * West region is the most profitable.
  * Discounts above 30% reduce profitability significantly.
